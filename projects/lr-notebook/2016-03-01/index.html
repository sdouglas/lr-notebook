Worked on folder list formatting. The best way to implement a folder index would probably be to write a plugin that builds virtual paths based on the database contents for specific record paths. Instead, I built a poor man's folder index by copying the lektor categories example. We thus explicitly define "blessed" folder names as children of the folders model, make it easy to create new names, or delete unused names.
